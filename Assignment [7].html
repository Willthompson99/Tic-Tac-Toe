Tic Tac Toe Game

Overview

The Tic Tac Toe game assignment involves creating a simple graphical Tic Tac Toe game using Java and the Swing framework. The game is structured within the TicTacToe class, which extends JFrame and implements ActionListener. The user interface includes a 3x3 grid of buttons for the game board, a label to display the current player's turn, and dialog boxes to input player names. Players take turns clicking on the grid buttons to mark their moves. The game checks for a win or draw condition after each move and displays an appropriate message using JOptionPane. The game ends when a player wins or all cells are filled, resulting in a draw. The program demonstrates key aspects of event-driven programming, user interaction, and basic game logic implementation in Java.
